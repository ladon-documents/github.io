Websocket API

Live updates are developed using the Websockets API.

Updates can be invoked by the frontend as well as the backend. The concept is that live updates are sent by the backend continuously to connected frontend clients and initial state is requested after connect of a new client.

To handle different events and messages each message contains a separate selector field and the payload.

{
  "selector": "ldn.docs.action.list"
}
  • ldn - ladon namespace
  • docs - message about documents
  • action - action request from the frontend
  • list - name of requested action - list all documents

Document Selectors

ldn.docs.action.list​

Description​

Initialize or load a document view

Request​

{
  "selector": "ldn.docs.action.list",
  "payload": "/mydocs/"
}

Response​

{
  "selector": "ldn.docs.added",
  "payload": {
    "content-type": "application/json",
    "created": "2023-01-01T00:00:00",
    "last-modified": "2023-01-01T00:00",
    "bucket": "mydocs",
    "key": "config/",
    "path": "/mydocs/config/",
    "etag": "",
    "metadata": {},
    "owner": "admin",
    "size": 0,
    "version": "none"
  }
}

Task Selectors

ldn.tasks.action.start​

Description​

Start a new task

Request​

{
  "selector": "ldn.tasks.action.start",
  "payload": {
    "context": "documents",
    "task-name": "Ladon Copy",
    "params" : {
      "sources": ["/mydocs/"],
      "target": "/mydocs2/"
    }
  }
}

Response​

{
  "selector": "ldn.tasks.added",
  "payload":{
    "uid": "958def24-2d91-47e0-a9b7-badac2510f3c",
    "context" : "documents",
    "task-name": "Ladon Copy Documents",
    "state": "running",
    "progress": 34,
    "eta": 46
  }
}

ldn.tasks.action.cancel​

Description​

Cancel a task

Request​

{
  "selector": "ldn.tasks.action.cancel",
  "payload":  "958def24-2d91-47e0-a9b7-badac2510f3c"
}

Response​

{
  "selector": "ldn.tasks.added",
  "payload":{
    "uid": "958def24-2d91-47e0-a9b7-badac2510f3c",
    "context" : "documents",
    "task-name": "Ladon Copy Documents",
    "state": "canceled",
    "progress": 34,
    "eta": 0
  }
}

ldn.tasks.action.list​

Description​

List current tasks

Request​

{
